HTML Images

An image can be shown using <img> tag along with its src attribute.

< img src="https://media.4-paws.org/1/e/d/6/1ed6da75afe37d82757142dc7c6633a532f53a7d/VIER%20PFOTEN_2019-03-15_001-2886x1999-1920x1330.jpg" >

Valid image extensions are the Following:

  1. png
  2. jpg
  3. gif
  4. svg
  5. webp
